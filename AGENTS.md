# AGENTS.md

这个项目是一个静态个人博客网站，用来展示个人介绍、学习记录、项目和博客文章。

## 维护规则

1. 只允许使用 HTML、CSS、JavaScript。
2. 不要引入 React、Vue、Next.js、数据库或后端服务。
3. 不要把项目改成需要构建工具才能运行的网站。
4. 修改前先用通俗中文说明计划，让用户知道准备改哪里、为什么改。
5. 修改后用通俗中文说明具体改了哪些文件。
6. 每次改动后都要检查页面链接是否正确，尤其是首页到博客文章、文章页返回首页、CSS 和 JavaScript 的路径。
7. 每次改动后都要检查手机端显示，确保页面不会横向溢出，文字和卡片不会挤在一起。
8. 文案要自然、真实，适合学生个人网站，不要写虚假的工作经验或夸张介绍。
9. 不要加入用户没有提供的隐私信息。
10. 保持文件结构简单，方便新手理解和继续修改。

## 当前文件结构

- `index.html`：网站首页。
- `style.css`：网站样式，包括浅色/深色模式和手机端适配。
- `script.js`：网站交互，目前主要负责深色模式切换。
- `.nojekyll`：让 GitHub Pages 按普通静态网站发布，不使用 Jekyll 处理。
- `README.md`：给用户阅读的项目说明和预览方法。
- `CHATGPT_PROJECT_CONTEXT.md`：给 ChatGPT 理解项目用的说明。
- `posts/`：博客文章页面所在文件夹。

## 修改后的检查清单

每次修改后，请至少检查：

1. 首页能正常打开。
2. CSS 能正常加载。
3. JavaScript 能正常加载。
4. 博客文章卡片能打开对应文章。
5. 文章页面能返回首页。
6. 手机屏幕下没有横向滚动。
7. 深色模式按钮仍然可用。
